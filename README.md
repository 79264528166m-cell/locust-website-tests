# Проект нагрузочного тестирования с Locust и Faker

Учебный проект по автоматизации нагрузочного тестирования с генерацией синтетических данных.

## 📋 Описание

Проект демонстрирует использование библиотек Locust и Faker для создания нагрузочных тестов API интернет-магазина. Виртуальные пользователи выполняют различные действия: просмотр товаров, добавление в корзину, оформление заказов с уникальными данными.

## 🛠 Технологии

- Python 3.8+
- Locust - фреймворк для нагрузочного тестирования
- Faker - генерация синтетических данных
- pytest - тестирование (опционально)

## 🚀 Установка и запуск

1. Клонировать репозиторий:
```bash
git clone https://github.com/ВАШ_ЛОГИН/locust-faker-loadtester.git
cd locust-faker-loadtester
2. Создать виртуальное окружение:
python -m venv venv
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
3. Установить зависимости:
pip install locust faker
4. Запустить тесты:
locust
5. Открыть браузер: http://localhost:8089
